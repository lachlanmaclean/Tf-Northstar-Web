# Northstar Web Server Browser

## Requirements 

|Name | Version
|-----|--------
|Node |16.13.1+

Install to local machine with 

```
git clone https://github.com/lachlanmaclean/Tf-Northstar-Web.git
cd Tf-Northstar-Web
npm install
```

This will install the application and its dependancies

## To run the app
```
node run start
```
